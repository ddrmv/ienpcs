Create PC

You can create a Player Character (PC), the main one of which is also known as CHARNAME. You can add one or more PC to a party. Most of the games have an option to easily add multiple custom created PCs to a party, while others were created with fully custom created party and no joinable NPCs in mind.

There are no restrictions for the non-numeric fields, except for the length of the text. The optional portrait image should be up to 100 KB.
